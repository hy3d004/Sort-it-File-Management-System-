# 3D 의뢰파일 자동화

## 자동 트리거: 에어테이블 행 붙여넣기

사용자가 탭으로 구분된 텍스트를 붙여넣으면 (숫자로 시작하고 날짜 형식 포함),
에어테이블 행으로 인식하고 아래 절차를 자동 실행한다.

### Step 1 — 파싱 확인
6번째 컬럼(업체명)과 9번째 컬럼(아이템명)을 추출해 사용자에게 확인받는다:
```
📋 파싱 결과:
   업체명: OOO
   아이템명: OOO

맞나요? (맞으면 "응" / 틀리면 수정해서 말씀해 주세요)
```

### Step 2 — 폴더 생성
확인 후 아래 명령어를 실행한다:
```bash
cd "c:/VSCODE/3DFILE" && PYTHONUTF8=1 "C:/Users/46510/AppData/Local/Programs/Python/Python312/python.exe" save_from_paste.py --paste "붙여넣은_전체_텍스트"
```

### Step 3 — 결과 보고
```
✅ 완료!
📁 YYMMDD_아이템명_업체명
```
