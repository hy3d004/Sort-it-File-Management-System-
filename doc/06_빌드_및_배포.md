# 6. 빌드 및 배포

## 6.1. 개발 환경 설정

### 6.1.1. 필수 패키지 설치
```bash
pip install PyQt6 Pillow pyinstaller
```

### 6.1.2. 개발 모드 실행
```bash
python app.py
```

## 6.2. EXE 빌드

### 6.2.1. 빌드 명령
```bash
pyinstaller --onefile --windowed --name "Sort it" --icon "assets/icon.ico" --add-data "assets;assets" app.py
```

| 옵션 | 설명 |
|---|---|
| `--onefile` | 단일 EXE 파일로 패키징 |
| `--windowed` | 콘솔 창 숨김 (GUI 전용) |
| `--name "Sort it"` | EXE 파일명 |
| `--icon "assets/icon.ico"` | 프로그램 아이콘 |
| `--add-data "assets;assets"` | assets 폴더를 EXE에 포함 |

### 6.2.2. 간편 빌드
```bash
build.bat
```
`build.bat`을 실행하면 PyInstaller 설치 확인 후 자동 빌드.

### 6.2.3. 빌드 결과
| 항목 | 값 |
|---|---|
| 출력 경로 | `dist/Sort it.exe` |
| 파일 크기 | 약 36MB |
| 포함 내용 | Python 런타임 + PyQt6 + 앱 코드 + assets |

## 6.3. 아이콘 변환
PNG → ICO 변환이 필요한 경우:
```python
from PIL import Image
img = Image.open("assets/icon.png").convert("RGBA")
sizes = [(16,16),(32,32),(48,48),(64,64),(128,128),(256,256)]
img.save("assets/icon.ico", format="ICO", sizes=sizes)
```

## 6.4. 배포

### 6.4.1. 배포 파일
`dist/Sort it.exe` 단일 파일만 전달하면 됩니다. 받는 사람 PC에 별도 설치 불필요.

### 6.4.2. 배포 방법

| 방법 | 설명 |
|---|---|
| 사내 공유폴더 | Z: 드라이브 공용 폴더에 업로드 |
| 카카오톡/메일 | `.zip`으로 압축 후 전송 (EXE 직접 전송 차단됨) |
| USB | 파일 복사 |
| GitHub Releases | 레포지토리에 릴리스 태그 생성 후 EXE 업로드 |

### 6.4.3. 실행 조건
| 조건 | 설명 |
|---|---|
| OS | Windows 10 이상 |
| Z: 드라이브 | 사내 네트워크 드라이브 매핑 필요 |
| 권한 | 대상 폴더 쓰기 권한 |
| 기타 | Python, PyQt6 등 별도 설치 불필요 |

## 6.5. 오류 대응

| 오류 | 원인 | 해결 |
|---|---|---|
| `드라이브에 접근할 수 없습니다` | Z: 드라이브 미연결 | 네트워크 드라이브 연결 확인 |
| `날짜, 업체명, 아이템명을 모두 입력하세요` | 파싱 안 됨 또는 필드 비어있음 | 에어테이블 행 다시 붙여넣기 |
| `탭 구분 컬럼 부족` | 복사한 텍스트가 에어테이블 행이 아님 | 에어테이블에서 행 선택 후 Ctrl+C 확인 |
| 파일 복사 오류 | 파일 접근 권한 또는 디스크 용량 부족 | 원본 파일 경로 및 디스크 확인 |
| 프로그램 미실행 | Windows Defender/백신이 EXE 차단 | 예외 등록 또는 허용 |
