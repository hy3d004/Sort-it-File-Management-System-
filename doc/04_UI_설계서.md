## 4.2. 위젯 목록

| 위젯 | 클래스 | ObjectName | 설명 |
|---|---|---|---|
| 탭 헤더 | QFrame | `tabBar` | 다크 네이비 배경, 탭 버튼 + 로고 |
| 탭 버튼 | QPushButton | `tabBtn` | 체크 가능, 활성 탭에 파란 하단 보더 |
| 붙여넣기 영역 | PasteTextEdit | `pasteArea` | QTextEdit 상속, Ctrl+V 시 기존 텍스트 자동 클리어 |
| 초기화 버튼 | QPushButton | `chipBtn` | 알약형 둥근 버튼 |
| 섹션 라벨 | QLabel | `sectionLabel` | 대문자, 작은 글씨, 회색 |
| 파싱 키 | QLabel | `parseKey` | 우측 정렬, 회색 |
| 파싱 값 | QLineEdit | `parseValue` | 편집 가능, filled 속성으로 스타일 변경 |
| 폴더 미리보기 | QLabel | `folderPreview` | Consolas 폰트, 회색 |
| 파일 수 뱃지 | QLabel | `badge` | 파란 알약형 |
| 드롭 영역 | DropArea(QFrame) | `dropArea` | 점선 테두리, 드래그앤드롭 + 클릭 지원 |
| 파일 리스트 | QListWidget | `fileList` | 커스텀 아이템 위젯 사용 |
| 파일 아이템 | FileItemWidget(QFrame) | `fileItem` | 확장자 뱃지 + 이름 + 크기 + 삭제 |
| 확장자 뱃지 | QLabel | `extBadge` | 3D 파일은 초록, 일반은 파랑 |
| 파일 추가 버튼 | QPushButton | `btnSmall` | 호버 시 파란색 |
| 삭제 버튼 | QPushButton | `btnSmallDel` | 호버 시 빨간색 |
| CTA 버튼 | QPushButton | `ctaBtn` | 다크 네이비, 탭별 다른 텍스트 |
| 상태 점 | QLabel | `sDot` | 초록/회색 전환 |
| 상태 텍스트 | QLabel | `sTxt` | 현재 상태 메시지 |
