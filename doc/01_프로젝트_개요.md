# 1. 프로젝트 개요

## 1.1. 프로젝트 정보
| 항목 | 내용 |
|---|---|
| 프로젝트명 | Sort it — 폴더 생성 프로그램 |
| 버전 | v2.1 |
| 개발 언어 | Python 3.14 |
| UI 프레임워크 | PyQt6 |
| 배포 형태 | PyInstaller 단일 EXE (36MB) |
| 대상 사용자 | 한양3D팩토리 제품개발팀 실무자 |
| GitHub | https://github.com/hy3d004/Sort-it-File-Management-System- |

## 1.2. 프로젝트 목적
에어테이블에서 관리하는 3D 의뢰 데이터를 기반으로, 사내 네트워크 드라이브(Z:)에 **규격화된 폴더를 자동 생성**하고 관련 파일을 분류·저장하는 데스크톱 프로그램.

## 1.3. 해결하는 문제
- 수작업으로 폴더를 만들고 파일을 옮기는 반복 업무 제거
- 폴더명 오타, 날짜 형식 불일치 등 **휴먼 에러 방지**
- 3D 파일과 이미지 파일의 저장 경로가 달라 혼선 발생 → **탭 분리로 해결**

## 1.4. 기술 스택
| 구분 | 기술 |
|---|---|
| 언어 | Python 3.14 |
| GUI | PyQt6 (QSS 커스텀 스타일링) |
| 파일 처리 | shutil, pathlib |
| 비동기 처리 | QThread (파일 복사 시 UI 멈춤 방지) |
| 빌드 | PyInstaller --onefile |
| 버전 관리 | Git + GitHub |
